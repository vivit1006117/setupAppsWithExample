AOT JIT
Ahead-of-time Just-in-time
The Angular ahead-of-time (AOT) compiler converts your Angular HTML and TypeScript code into efficient JavaScript code during the build phase before the browser downloads and runs that code. The Angular Just-in-Time (JIT) compiler converts your Angular HTML and TypeScript code into efficient JavaScript code at run time, as part of bootstrapping. JIT compilation is the default when you run Angular's ng build and ng serve CLI commands, and is a good choice during development.
This is the best compilation mode for production environments, with decreased load time and increased performance. JIT mode is strongly discouraged for production use because it results in large application payloads that hinder the bootstrap performance.
